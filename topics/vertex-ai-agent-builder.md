# Vertex AI Agent Builder

> _Question to answer in my own words: What is Vertex AI Agent Builder?_

## Answer

Vertex AI Agent Builder is Google Cloud's platform for building, deploying, and managing AI agents on Vertex AI, primarily using Gemini models. It bundles together the tools needed for production agents: the Agent Development Kit (ADK) for defining agent logic in code, the Agent Engine as a managed runtime that deploys and scales agents for you, and built-in grounding and retrieval so agents can answer from your own enterprise data and search. Because it lives inside Google Cloud, it also brings enterprise concerns like security, monitoring, and access controls. It's aimed at teams that want to build conversational and agentic applications grounded in their data without operating all the underlying infrastructure themselves.
