# LanceDB

> _Question to answer in my own words: What is LanceDB?_

## Answer

LanceDB is an open-source, embedded vector database that runs in-process within your application, much like SQLite does for relational data, so there is no separate server to operate. It is built on the Lance columnar data format, which lets it store vectors and their metadata efficiently on local disk or object storage while supporting fast queries. LanceDB is designed to handle multimodal data such as text, images, and video, and it integrates with common AI frameworks. Its serverless, file-based design makes it a lightweight and cost-effective option for both prototypes and production workloads that benefit from keeping data close to the application.
