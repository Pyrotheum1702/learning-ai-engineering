# Chroma

> _Question to answer in my own words: What is Chroma?_

## Answer

Chroma is an open-source, lightweight vector database designed to be simple and developer-friendly, making it a popular choice for prototyping and building AI applications. It can run fully in-memory for quick experiments or persist data to disk for longer-lived projects, and it embeds directly into your application rather than requiring a separate server to manage. Chroma handles the storage, indexing, and similarity search of embeddings, and it integrates cleanly with common AI frameworks like LangChain and LlamaIndex. Its low setup cost and Python/JavaScript clients make it a common starting point for RAG and semantic search prototypes.
