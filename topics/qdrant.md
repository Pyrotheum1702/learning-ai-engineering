# Qdrant

> _Question to answer in my own words: What is Qdrant?_

## Answer

Qdrant is an open-source vector database and similarity search engine written in Rust, which gives it strong performance and memory efficiency. It uses HNSW indexing for fast approximate nearest neighbor search and offers rich payload (metadata) filtering, so results can be narrowed by structured conditions alongside vector similarity. Qdrant exposes both REST and gRPC APIs, can be self-hosted or run as a managed cloud service, and includes production features like snapshots, sharding, and replication. Its balance of speed, filtering, and operational maturity makes it a popular choice for production semantic search and RAG systems.
