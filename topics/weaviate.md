# Weaviate

> _Question to answer in my own words: What is Weaviate?_

## Answer

Weaviate is an open-source vector database that can be self-hosted or used as a managed cloud service. It is notable for its built-in vectorization modules, which can generate embeddings from your data automatically using model providers, and for its support of hybrid search that combines vector similarity with traditional keyword (BM25) search for better relevance. Data in Weaviate is organized into a schema of classes and objects, and it can be queried through GraphQL or REST APIs. Its flexibility and hybrid capabilities make it a strong fit for semantic search, RAG, and applications that need both meaning-based and exact-match retrieval.
