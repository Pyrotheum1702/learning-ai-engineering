# OpenAI AgentKit & Agent SDK

> _Question to answer in my own words: What are OpenAI's AgentKit and Agents SDK?_

## Answer

AgentKit and the Agents SDK are OpenAI's tooling for building agents on their models. The **Agents SDK** is a lightweight, open-source framework for orchestrating agentic workflows in code — it provides core primitives like agents, tools, handoffs between agents, guardrails, sessions, and built-in tracing, and it runs the agent loop for you. **AgentKit** is the broader set of building blocks around it, including a visual Agent Builder for composing workflows on a canvas, connectors to data sources, and evaluation tooling. Together they let you go from prototyping an agent visually to deploying a production, multi-agent system on OpenAI's stack.
